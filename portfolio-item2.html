Routine Alarm Clock OVERVIEW

Routine Alarm full stack application that allows a user to create a morning 'routine' consisting of 'tasks' with custom duration time values created by the user. The user will be given the best time to set their alarm based on their routine and when they would like to arrive at a destination like work or school.
