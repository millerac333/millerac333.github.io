Ball-Up OVERVIEW

Ball-Up gives the ability for Ballers (someone, anyone) to find public outdoor and indoor basketball courts in the local Nashville area as well as create pick-up games at specific basketball courts. Another user of the application is able to see active and upcoming pick-up games at the specified court; that user is able to "join" the pick-up game at that court location or any pick-up game at any of the listed courts.
